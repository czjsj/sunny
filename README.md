# 开源项目和模块

本项目使用了以下开源项目和模块，以下是主要依赖项的列表。这些开源项目为项目的开发和功能实现提供了重要支持。

## 主要依赖项 (Dependencies)

- **@antv/g2plot** (^2.3.17): AntV G2Plot 是一个基于 G2 的图表库，提供丰富的图表类型和交互功能。
- **@jiaminghi/data-view** (^2.10.0): 数据可视化组件库，用于创建数据大屏。
- **axios** (0.18.1): HTTP 客户端，用于发送异步请求。
- **clipboard** (2.0.4): 现代化的剪贴板 API 库。
- **codemirror** (5.45.0): 代码编辑器组件。
- **core-js** (^3.20.2): JavaScript 标准库的 polyfill。
- **d3-geo** (^3.0.1): D3.js 的地理投影库。
- **driver.js** (0.9.5): 用户引导库，用于创建交互式教程。
- **dropzone** (5.5.1): 文件上传组件。
- **echarts** (4.2.1): 百度 ECharts 图表库。
- **element-ui** (2.13.2): Vue.js 的 UI 组件库。
- **file-saver** (2.0.1): 文件保存库。
- **fuse.js** (3.4.4): 轻量级模糊搜索库。
- **js-cookie** (2.2.0): JavaScript Cookie 操作库。
- **jsonlint** (1.6.3): JSON 语法检查器。
- **jszip** (3.2.1): JavaScript ZIP 文件处理库。
- **lodash** (^4.17.21): JavaScript 工具库。
- **lodash.throttle** (^4.1.1): Lodash 的节流函数。
- **moment** (^2.29.1): JavaScript 日期处理库。
- **normalize.css** (7.0.0): CSS 重置样式表。
- **nprogress** (0.2.0): 进度条库。
- **path-to-regexp** (2.4.0): 路径匹配库。
- **screenfull** (4.2.0): 全屏 API 库。
- **script-loader** (0.7.2): Webpack 脚本加载器。
- **sortablejs** (1.8.4): 拖拽排序库。
- **three** (^0.133.1): Three.js 3D 图形库。
- **tui-editor** (1.4.10): TOAST UI Editor 富文本编辑器。
- **vue** (2.6.10): Vue.js 框架。
- **vue-count-to** (1.0.13): Vue 数字动画组件。
- **vue-d3-geo** (^0.4.6): Vue D3 地理组件。
- **vue-digital-transform** (^1.1.1): Vue 数字变换组件。
- **vue-router** (3.0.2): Vue.js 路由库。
- **vue-splitpane** (1.0.4): Vue 分割面板组件。
- **vuedraggable** (2.20.0): Vue 拖拽组件。
- **vuex** (3.1.0): Vue.js 状态管理库。
- **xlsx** (0.14.1): Excel 文件处理库。
